Este programa resolve o exercício Cash, proposto na Semana 1 do curso CS50 de Harvard.
O objetivo é calcular o menor número de moedas necessárias para devolver um valor de troco, informado pelo usuário em centavos.
O programa valida a entrada e utiliza uma estratégia gulosa (greedy), priorizando moedas de maior valor (25, 10, 5 e 1 centavo).
